# Objective #1 - GRID class and tab system.

## Grid class

Your first objective is to impement the logic in `mock/grid.js`. What I want you to do is use that mockup class to create a class based system for GRID instances.

## Tab system

Implement a "tab" system, similar to that of VScode, execpt instead of individual tabs for editor instances, individual tabs will contain individual GRID instances/windows. Tabs should have a title and an "x" button next to them and should be aligned to the left, and be responsive when other tabs are present.
